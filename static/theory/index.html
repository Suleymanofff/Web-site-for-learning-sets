Теория: Ассоциативное свойство множеств

Что такое множество?

Множество — совокупность объектов, называемых элементами множества. Например, A = {1, 2, 3}.

Операции над множествами

  • Объединение (A ∪ B) — все элементы, принадлежащие A или B.
  • Пересечение (A ∩ B) — элементы, принадлежащие одновременно A и B.
  • Разность (A \ B) — элементы, принадлежащие A, но не B.

Ассоциативное свойство

Результат операции не зависит от порядка группировки, если операции одинаковы.

Объединение

(A ∪ B) ∪ C = A ∪ (B ∪ C)

Пример:

A = {1,2}, B = {2,3}, C = {3,4}

(A ∪ B) ∪ C = {1,2,3,4}

A ∪ (B ∪ C) = {1,2,3,4}

Пересечение

(A ∩ B) ∩ C = A ∩ (B ∩ C)

Пример:

A = {1,2,3}, B = {2,3,4}, C = {3,4,5}

(A ∩ B) ∩ C = {3}

A ∩ (B ∩ C) = {3}

Зачем это важно?

  • Упрощает выражения и группировку операций.
  • Делает алгоритмы гибче.
  • Облегчает доказательства и вычисления.
